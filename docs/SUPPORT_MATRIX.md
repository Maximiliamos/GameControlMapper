# Support matrix

The RC is awaiting human validation. Automated coverage does not by itself grant `Supported` status.

| Area | RC status |
|---|---|
| Core touch, focus, profiles, diagnostics | AutomatedOnly — awaiting guided validation |
| Mixed DPI, multiple monitors, negative origin | AutomatedOnly — requires matching hardware validation |
| XInput, Macro/Sequence, Android/ADB, RawInput, Interception, ViGEm, pinch, rotation | Unsupported |
| Specific games, emulators and anti-cheat systems | No compatibility guarantee |
