# Privacy

Game Control Mapper has no telemetry and sends no data over the network. Profiles, logs, backups and diagnostic exports remain local. Diagnostic ZIP files intentionally exclude profile contents, pressed-key history, process lists and unrelated window titles. Review an export before sharing it.

To remove the application, stop mapping, delete the extracted program directory, then delete its local `Profiles`, `Logs`, crash-report and diagnostic-export files if they were stored elsewhere. No installer service or auto-updater is present.
